# Algoritmo Guloso Simples de Escalonamento em Redes mmWave

## 📝 Projeto
Este projeto se trata de um algoritmo Guloso simples para encontrar um escalonamento ótimo de dispositivos em redes mmWave.

## 📑 Organização do projeto
### 🗂️ Pastas
- **BILPCompEntries**: Entradas a serem comparadas com o BILP

### 📰 Arquivos
- **main.py**: Script Python que executa o algoritmo principal.
  - **Uso**: apenas executar _main.py_ e inserir os dados. O resultado será mostrado na tela.

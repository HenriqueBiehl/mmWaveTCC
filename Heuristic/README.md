# Algoritmo Heurístico de Escalonamento em Redes mmWave

## 📝 Projeto
Este projeto se trata de um algoritmo Heurístico para encontrar um escalonamento ótimo de dispositivos em redes mmWave. Trata-se de uma Heurística Gulosa aplicada ao problema.

## 📑 Organização do projeto
### 📰 Arquivos
- **main.py**: Script Python que executa o algoritmo principal.
  - **Uso**: apenas executar _main.py_ e inserir os dados. O resultado será mostrado na tela.
- **heuristic_scheduler.py**: Arquivo com as funções principais que são utilizadas pelo algoritmo
- **ent1.txt**: Entrada de exemplo para o programa

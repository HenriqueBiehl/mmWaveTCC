# Algoritmo Gerador de Rates para Algoritmos de Escalonamento em Redes mmWave

## 📝 Projeto
Este projeto se trata de um algoritmo gerador de rates de dispostivos para utilizar como entradas nos algoritmos de escalonamento em redes mmWave.

## 📑 Organização do projeto
### 📰 Arquivos
- **main.py**: Script Python que executa o algoritmo principal.
  - **Uso**: apenas executar _main.py_ e inserir os dados. O resultado será mostrado na tela.
- **output.txt**: Saída do programa, pronta para ser usada como entrada nos algoritmos principais
- **rates.txt**: Entrada para o programa gerador de rates
- **rates_2.txt**: Entrada para o programa gerador de rates
- **rates_entry.txt**: Exemplo de entrada do programa, mostrando quais dados devem ser introduzidos
